## Что такое Content Bot

Content Bot — система автоматической публикации контента в каналы MAX (max.ru).

**Принцип работы:**
1. Бот парсит посты из указанных Telegram-каналов или RSS-лент
2. Фильтрует нежелательный контент
3. При необходимости переписывает текст через нейросеть
4. Публикует посты в MAX-каналы по расписанию

---

## Запуск бота

Найди бота в Telegram по токену или имени и напиши `/start`.

Если у тебя нет доступа — обратись к суперадмину.

---

## Главное меню

После `/start` открывается главное меню с кнопками:

| Кнопка | Раздел |
|--------|--------|
| 📊 Статистика | Сколько постов опубликовано |
| 🔔 Уведомления | Настройка оповещений |
| 📺 Каналы & Сетки | Управление каналами и группами |
| 🔗 Источники | Откуда берём контент |
| ⏰ Расписание | Когда публиковать |
| 🚫 Фильтры | Что блокировать |
| ✍️ Рерайт | Переписывание текстов |
| 🤖 Нейросети | API ключи и модели |
| 👥 Администраторы | Управление доступом (только суперадмин) |

> На каждом экране есть кнопки **◀️ Назад** и **🏠 Главная** для навигации.
> Кнопка **❌ Отмена** появляется когда нужно ввести текст — нажми её чтобы вернуться назад без сохранения.

## Роли и права доступа

### 👑 Superadmin
- Полный доступ ко всем функциям
- Единственный кто может управлять администраторами
- Нельзя удалить

### 🔧 Admin
- Управление каналами, сетками, источниками
- Настройка расписания и фильтров
- Управление рерайтом и нейросетями
- Просмотр статистики

### 👁 Viewer
- Только просмотр всех разделов
- Не может вносить изменения

## 📊 Статистика

Показывает сколько постов опубликовано по каждому каналу.

**Кнопки:**
- **📅 За неделю** — статистика за последние 7 дней
- **🔗 По источникам** — топ-20 источников по количеству постов

**Что означают числа:**
- `сегодня N` — опубликовано сегодня
- `всего N` — всего опубликовано за всё время
- `очередь N` — ждут публикации

## 📺 Каналы и Сетки

### Сетки

**Сетка** — группа каналов объединённых по теме (например "Города России", "Контент").

**Создать сетку:**
1. Нажми **➕ Новая сетка**
2. Введи название (латиницей, например `sport` или `novosti`)
3. Нажми **➕ Добавить каналы** → введи ключи каналов через запятую

**Добавить каналы в сетку пачкой:**
```
kazan
nizhny_novgorod
samara
ufa
```
или через запятую: `kazan, nizhny_novgorod, samara`

**Удалить сетку:** нажми 🗑 рядом с названием → подтверди.

---

### Все каналы

Список всех каналов с пагинацией (15 на странице).

**Иконки статуса:**
- 🟢 — канал активен, посты публикуются
- 🔴 — канал на паузе

**Кнопки у каждого канала:**
- **⏸/▶️** — поставить на паузу или возобновить
- **⚙️ Название** — настройки этого канала
- **🗑** — удалить канал (с подтверждением)

**Добавить новый канал:**
1. Нажми **➕ Добавить канал**
2. Введи ключ (латиницей, без пробелов, например `moscow`)
3. Введи MAX chat_id (отрицательное число, например `-72819209448869`)

---

## 🔗 Источники

Здесь указываешь откуда брать контент для каждого канала.

**Иконки:**
- ✅ — есть хотя бы один источник
- ➕ — источников нет

**Добавить источники:**
1. Выбери канал из списка
2. Нажми **➕ Добавить**
3. Введи источники:

```
# Telegram-каналы (без @):
abakan_novosti
news_abakan
khakassia_news

# RSS-ленты сайтов:
https://abakan-news.ru/feed/
https://news.ru/rss
```

Можно добавлять сразу несколько через запятую или с новой строки.

**Типы источников:**
- **Telegram** — username канала без @
- **Web/RSS** — ссылка начинается с `https://`

**Проверить источники:** нажми **✅ Проверить источники** — бот пингует каждый и показывает доступен ли он.

**Удалить источник:** нажми **🗑 Удалить** → выбери нужный из списка.

## ⏰ Расписание

Настройка когда публиковать посты.

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| Начало | 8:00 | Первая публикация дня |
| Конец | 22:00 | Последняя публикация дня |
| Интервал | каждые 2 ч | Как часто публиковать |
| Ночная пауза | ✅ Вкл | Не публиковать ночью |

**Изменить интервал:** нажми **⏱ Интервал** → введи число от 1 до 12.

**Пример:** начало 9, конец 21, интервал 3 → публикации в 9:00, 12:00, 15:00, 18:00, 21:00.

## 🚫 Фильтры

Три уровня фильтрации:

### 🌐 Глобальные фильтры
Применяются ко **всем** каналам без исключения.

**Переключатели (✅/❌):**
| Фильтр | Описание |
|--------|----------|
| 📝 Требовать текст | Пропускать посты без текста (только картинка) |
| 🇬🇧 Блок. английский | Пропускать посты где >50% букв латинские |
| 🤬 Блок. мат | Пропускать посты с нецензурной лексикой |
| 📢 Блок. CAPS LOCK | Пропускать посты где >50% заглавных букв |
| 😶 Блок. только эмодзи | Пропускать посты из одних эмодзи |
| 🔗 Блок. ссылки | Пропускать посты со ссылками |

**Числовые настройки:**
- **📏 Мин. символов** — посты короче этого числа пропускаются (0 = выкл)
- **📐 Макс. символов** — посты длиннее пропускаются (0 = без ограничений)
- **🔤 Мин. слов** — минимум слов в посте
- **⏳ Макс. возраст** — не брать посты старше N часов (0 = без ограничений)

**Списки слов:**
- **🚫 Стоп-слова** — посты с этими словами блокируются
- **✅ Белый список** — если не пустой, публикуются ТОЛЬКО посты с этими словами
- **⚙️ Regex-паттерны** — сложные правила (для продвинутых)

---

### 🗂 Фильтры по сетке
Настройки для конкретной сетки каналов.

**Дополнительные переключатели:**
| Фильтр | Описание |
|--------|----------|
| 📷 Только с фото | Публиковать только посты с изображениями |
| 🎥 Только с видео | Публиковать только посты с видео |
| 🖼 Требовать медиа | Посты без фото/видео пропускаются |
| 🔄 Блок. репосты | Только оригинальные посты (не пересланные) |
| 🗓 Блок. сезонные | Блокировать праздничные поздравления не в сезон |
| 🛒 Блок. маркетплейсы | Блокировать упоминания WB, Ozon и т.д. |

---

### ⭐ Приоритетные слова
Посты содержащие эти слова публикуются **первыми** в очереди.

Добавляй слова связанные с важными темами: `срочно, экстренно, важно`.

## ✍️ Рерайт

Автоматическое переписывание текстов постов перед публикацией.

**Настройки:**
- **▶️/⏸ Вкл/Выкл** — включить или отключить рерайт
- **🤖 Groq / 🧠 Claude** — выбор модели нейросети
- **📝 Свой промпт** — задать индивидуальный стиль переписывания
- **🔄 Сбросить промпт** — вернуть стандартный промпт

**Пример своего промпта:**
```
Перепиши новость коротко и ёмко для городского канала.
Добавь подходящий эмодзи в начало. Текст: {text}
```

> Используй `{text}` для вставки оригинального текста и `{topic}` для темы канала.

## 🤖 Нейросети

Управление API ключами для нейросетей используемых в рерайте.

**Доступные провайдеры:**
| Провайдер | Где взять ключ |
|-----------|---------------|
| ⚡ Groq (Llama) | console.groq.com |
| 🧠 Claude (Anthropic) | console.anthropic.com |
| 🟢 OpenAI (GPT) | platform.openai.com |
| 💎 Google Gemini | aistudio.google.com |

**Статус:**
- ✅ — ключ введён
- ❌ — ключ не задан
- 🔵 — этот провайдер активен для рерайта

**Как добавить ключ:**
1. Выбери провайдера
2. Нажми **🔑 Изменить ключ**
3. Вставь API ключ
4. Нажми **🔵 Сделать активной**

## 🔔 Уведомления

Бот присылает уведомления если что-то идёт не так.

**Настройки:**
- **▶️/⏸ Вкл/Выкл** — включить или отключить уведомления
- **⏱ Порог молчания** — через сколько часов без постов прислать сигнал (1-24 ч)
- **🌙 Ночной режим** — сколько ночных часов не беспокоить (0-12 ч)

## 👥 Администраторы

> Раздел доступен только суперадмину.

**Добавить администратора:**
1. Нажми **➕ Добавить**
2. Введи Telegram ID и роль: `123456789 admin`
3. Подтверди действие

**Роли при добавлении:**
- `admin` — полный доступ кроме управления администраторами
- `viewer` — только просмотр

**Изменить роль:** нажми **→ admin** или **→ viewer** рядом с пользователем.

**Удалить:** нажми **🗑 ID** → подтверди.

> Суперадмина удалить нельзя.

## ⚙️ Настройки канала

Индивидуальные настройки для каждого канала — переопределяют настройки сетки.

**Открыть:** раздел **📺 Каналы & Сетки** → **📋 Все каналы** → нажми **⚙️ Название канала**.

### 📹 Тип медиа
| Режим | Что публикуется |
|-------|----------------|
| 🖼+🎥 Любые медиа | Фото и видео без ограничений |
| 📷 Только фото | Только посты с изображениями |
| 🎥 Только видео | Только посты с видео |
| 📝 Только текст | Посты без медиа |
| 🖼 Медиа обязательно | Пропускать текстовые посты |

### 📏 Мин./Макс. длина
- Мин. символов — посты короче пропускаются
- Макс. символов — посты длиннее пропускаются
- Мин. слов — минимум слов в тексте

### 🚫 Стоп-слова
Индивидуальные стоп-слова только для этого канала.

### ✍️ Рерайт
- **⚙️ Из сетки** — использовать настройки сетки
- **✅ Включить** — рерайт всегда включён для этого канала
- **❌ Выключить** — публиковать оригинальный текст
- **📝 Свой промпт** — индивидуальный стиль

### 🗑 Сбросить настройки
Удаляет все индивидуальные настройки — канал начнёт использовать настройки сетки.

## Частые вопросы

**❓ Почему посты не публикуются?**
- Проверь что канал не на паузе (🟢 = активен)
- Проверь что есть источники с постами
- Проверь расписание — возможно сейчас ночное время
- Посмотри статистику — может очередь пуста

**❓ Посты публикуются, но контент некачественный?**
- Настрой стоп-слова для блокировки нежелательного контента
- Включи белый список слов для тематической фильтрации
- Добавь мин. длину текста чтобы убрать короткие бессмысленные посты

**❓ Как добавить новый город?**
1. Создай канал в MAX
2. Добавь бота как администратора
3. В боте: **📺 Каналы & Сетки** → **📋 Все каналы** → **➕ Добавить канал**
4. Введи ключ и chat_id
5. Добавь канал в нужную сетку
6. Добавь источники

**❓ Как узнать chat_id канала MAX?**
Бот автоматически получает chat_id через API когда ты добавляешь его как администратора.

**❓ Как остановить публикации на время?**
Поставь канал на паузу: **📺 Каналы & Сетки** → **📋 Все каналы** → **⏸** рядом с каналом.

**❓ Разница между стоп-словами сетки и канала?**
- Стоп-слова **сетки** блокируют для всех каналов в сетке
- Стоп-слова **канала** блокируют только в этом канале
- Глобальные стоп-слова блокируют везде
