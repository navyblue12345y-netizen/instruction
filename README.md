## Первый запуск

1. Найдите бота в Telegram и отправьте `/start`
2. Откроется **Главное меню** с кнопками разделов
3. Всё управление — только через кнопки, никаких команд вводить не нужно

> ⚠️ Бот принимает команды только от авторизованных пользователей.  
> Если вы видите "⛔" — обратитесь к суперадмину для получения доступа.

## 📊 Статистика

Показывает сколько постов опубликовано, сгруппировано по сеткам.

### Статистика за сегодня
Главная страница раздела. Показывает по каждой сетке:
- Название канала
- Постов сегодня
- Всего постов за всё время
- Постов в очереди (ещё не опубликованы)
- Итого по сетке за сегодня

### 📅 За неделю
Сводка по всем каналам за последние 7 дней.

### 🔗 По источникам
Топ-20 Telegram-каналов источников по количеству опубликованных постов.

## 📺 Каналы и Сетки

Основной раздел управления структурой публикаций.

### Что такое сетка
**Сетка** — группа каналов с общими настройками расписания.  
Например: сетка "Контент" (каналы про моду, вязание, дом) и сетка "Города России" (174 городских канала).

Каждая сетка работает **полностью независимо** от других:
- своё расписание публикаций
- свои источники контента
- свои фильтры
- свой режим работы (обычный или живой)

### Список сеток
На главной странице раздела отображаются все сетки с количеством каналов.  
Нажмите на сетку чтобы войти в неё.

### Внутри сетки

Показывает список каналов сетки с количеством источников у каждого.

**Кнопки управления:**
- **➕ Добавить каналы** — добавить каналы в сетку (можно несколько через запятую)
- **🗑 Удалить сетку** — удалить сетку (сами каналы остаются в системе)

**Для каждого канала:**
- **⏸ / ▶️** — поставить канал на паузу или возобновить
- **📋 Источники** — перейти к источникам канала
- **🗑** — удалить канал полностью (удаляется из всех мест)

### Создание новой сетки

1. Главная → 📺 Каналы & Сетки
2. Нажмите **➕ Новая сетка**
3. Введите название
4. Сетка создаётся автоматически с дефолтными настройками расписания и фильтров
5. Настройте расписание в разделе ⏰ Расписание

> ℹ️ При создании новой сетки она автоматически появляется в разделах ⏰ Расписание и 🚫 Фильтры — ничего делать вручную не нужно.

## 🔗 Источники

Источники — Telegram-каналы из которых бот берёт контент для публикации.

### Добавление источников

1. 📺 → Сетка → нажмите на канал → **📋 Источники**
2. Нажмите **➕ Добавить источники**
3. Введите username(ы) Telegram-каналов

Форматы ввода:
```
city_news_abakan
@city_news_abakan
city_news_abakan, another_channel
city_news_abakan
another_channel
(каждый с новой строки)
```

> Символ @ можно не ставить — бот добавит сам.

После добавления появятся кнопки:
- **➕ Добавить ещё** — продолжить добавлять источники
- **📋 К каналу** — вернуться к странице канала

### Удаление источника

В списке источников нажмите **🗑** рядом с нужным источником → подтвердите.

### Поддерживаемые типы источников

| Тип | Формат | Пример |
|-----|--------|--------|
| Telegram-канал | username без @ | `city_news` |
| RSS/Web | полный URL | `https://site.ru/feed/` |

## ⏰ Расписание

Управление временем публикаций. **Каждая сетка имеет своё независимое расписание.**

### Страница расписания

Показывает все сетки с текущими настройками:
- Режим (обычный или 🔴 живой)
- Время публикаций и интервал
- Количество постов в день
- Время сбора контента

### Настройки сетки

Нажмите **⚙️ [Название сетки]** для редактирования.

| Параметр | Описание |
|----------|----------|
| ⏱ Интервал публ. | Каждые N часов выходит один пост (1-24) |
| 📊 Постов/день | Сколько постов собирать за один ночной fetch |
| 🌅 Начало | С какого часа начинать публикации (МСК) |
| 🌆 Конец | До какого часа публиковать (МСК) |
| 🕐 Час fetch | В какое время собирать контент (обычно 23:00) |
| 🎬 Тип медиа | Тип медиа по умолчанию для всей сетки |
| ⏸/▶️ | Поставить всю сетку на паузу или включить |

### 🔴 Живой режим

Для новостных каналов — посты выходят сразу после появления в источнике.

**Как включить:**
1. ⏰ Расписание → ⚙️ [Сетка]
2. Нажмите **🔴 Вкл живой режим**
3. Нажмите **🔄 Применить сейчас** (обязательно!)

**Как работает:**
- Бот проверяет источники каждые N минут
- Нашёл новые посты → публикует сразу (до 3 за один цикл)
- Ничего нового → молчит, ждёт следующей проверки

**⏱ Интервал fetch (мин)** — как часто проверять (5-180 минут, рекомендуется 30).

### 🔄 Применить расписание

Кнопка внизу страницы расписания. Перезапускает движок публикаций с новыми настройками.

> ⚠️ После изменения расписания **обязательно** нажмите эту кнопку. Посты в очереди не теряются.

## 🚫 Фильтры

Фильтры определяют какие посты публиковать, а какие пропускать.

### Структура фильтров

```
🚫 Фильтры
├── 🌐 Глобальные        — для всех каналов всех сеток
├── ⭐ Приоритет          — слова для приоритетной публикации
├── 🗂 Контент           — фильтры для сетки "Контент"
│   ├── Настройки сетки
│   └── 📋 Каналы сетки
│       ├── moda → индивидуальные настройки
│       ├── dom  → индивидуальные настройки
│       └── vyazanie → индивидуальные настройки
└── 🗂 Города России     — фильтры для сетки "Города России"
    ├── Настройки сетки
    └── 📋 Каналы сетки (174 канала, пагинация по 15)
        └── abakan → индивидуальные настройки
```

### Приоритет фильтров

```
Настройки канала (высший приоритет)
    ↓ если не задано
Настройки сетки
    ↓ если не задано
Глобальные настройки (низший приоритет)
```

### Глобальные фильтры

Применяются ко всем каналам если не переопределены на уровне сетки или канала.

**Переключатели:**
- 🔞 Фильтр мата — блокировать посты с нецензурной лексикой
- 📢 Блокировать рекламу — фильтровать очевидную рекламу
- 🔗 Блокировать ссылки — посты со ссылками

**Числовые настройки:**
- 📏 Мин. символов — минимальная длина текста поста
- 📐 Макс. символов — максимальная длина (0 = без ограничений)
- 🔤 Мин. слов — минимальное количество слов
- ⏳ Макс. возраст — не брать посты старше N часов

**Списки слов:**
- 🚫 Стоп-слова — посты с этими словами пропускаются
- ✅ Белый список — если задан, публикуются ТОЛЬКО посты с этими словами
- ⚙️ Regex-паттерны — блокировка по регулярным выражениям

### Фильтры сетки

🚫 Фильтры → выберите сетку → настройки применяются ко всем каналам этой сетки.

Доступны те же параметры что и в глобальных, но только для одной сетки.

**📋 Каналы сетки** — список каналов с возможностью настройки каждого индивидуально.  
Каналы с индивидуальными настройками отмечены иконкой ⚙️.

### Настройки канала (через Фильтры)

🚫 Фильтры → Сетка → 📋 Каналы сетки → [Канал]

| Настройка | Описание |
|-----------|----------|
| 📹 Тип медиа | Любое / Только фото / Только видео / Только текст / Медиа обязательно |
| ✍️ Рерайт | Вкл / Выкл / Из сетки (наследовать) |
| 📏 Мин. длина | Минимальная длина текста для этого канала |
| 📐 Макс. длина | Максимальная длина текста |
| 🚫 Стоп-слова | Индивидуальные стоп-слова канала |
| ✅ Белый список | Индивидуальный белый список |
| 🗑 Сбросить | Удалить все индивидуальные настройки |

> 💡 **Пример:** для канала "Мода" можно поставить "Только видео", а для "Вязание" — "Любые медиа". Настройки не влияют на другие каналы.

## ✍️ Рерайт

Бот автоматически переписывает текст поста перед публикацией — создаёт короткую подпись с эмодзи (до 10 слов).

### Глобальные настройки рерайта

**Вкл/выкл** — включить или выключить рерайт для всей системы.

**Промпт** — инструкция для нейросети.  
По умолчанию: написать одно вдохновляющее предложение с эмодзи, до 10 слов, без хештегов.

**🔄 Сбросить промпт** — вернуть к заводскому.

### Рерайт для конкретного канала

Настраивается в 🚫 Фильтры → Сетка → Каналы → [Канал] → ✍️ Рерайт:
- **⚙️ Из сетки** — наследовать настройку от сетки
- **✅ Включить** — рерайт для этого канала всегда включён
- **❌ Выключить** — рерайт для этого канала всегда выключен

> 💡 Для новостных каналов рекомендуется **выключить рерайт** — новости публикуются с оригинальным текстом.

## 🔔 Уведомления

Бот присылает уведомление если канал долго ничего не публикует.

### Настройки

| Параметр | Описание |
|----------|----------|
| Вкл/выкл | Включить или выключить уведомления |
| ⏱ Порог молчания | Через сколько часов без постов прислать уведомление (1-24 ч) |
| 🌙 Ночной режим | Сколько часов ночью не беспокоить (с 23:00) |

**Рекомендации:**
- Порог молчания: 4-6 часов
- Ночной режим: 6 часов (не беспокоить с 23:00 до 05:00)

### Кто получает уведомления

Все пользователи с ролью **admin** и **superadmin**.

## Как всё работает изнутри

### Обычный режим (сетка "Контент")

```
23:00 МСК — Ночной сбор (Fetch)
  └── Заходит в Telegram-каналы из источников
  └── Берёт N свежих постов на каждый канал
  └── Проверяет фильтры (реклама, стоп-слова, длина, медиа)
  └── Переписывает текст через AI (если рерайт включён)
  └── Сохраняет в очередь

08:00, 10:00, 12:00 ... 22:00 МСК — Публикация
  └── Берёт 1 пост из очереди для каждого канала сетки
  └── Загружает медиа в MAX
  └── Публикует пост в канал
```

### Живой режим (сетка "Города России")

```
Каждые N минут:
  └── Смотрит новые посты в источниках
  └── Если есть новые → фильтрует → публикует сразу (до 3 за раз)
  └── Если нет новых → ничего не делает, ждёт следующей проверки
```

### Приоритет настроек

```
Настройки канала  ← высший приоритет (переопределяет всё)
       ↓
Настройки сетки   ← применяется если нет настроек канала
       ↓
Глобальные        ← базовый уровень (низший приоритет)
```

### Защита от дублей

Каждый пост имеет уникальный URL. Бот запоминает все URL которые видел за последние **30 дней** — повторная публикация одного поста исключена.

### Независимость сеток

Каждая сетка:
- Имеет явные настройки в конфиге (не наследует от других сеток)
- Запускается по своему расписанию независимо
- При изменении одной сетки другие не затрагиваются

## Частые вопросы

### Почему бот не публикует?

1. Проверьте что сетка активна: ⏰ Расписание → ⚙️ [Сетка] → статус
2. Проверьте что канал не на паузе: 📺 → Сетка → канал
3. Проверьте что есть источники: 🔗 Источники → канал
4. Проверьте что в очереди есть посты: 📊 Статистика → колонка "очередь"
5. Проверьте статус движка: ⚙️ Система

### Посты выходят с пустым текстом или коротким обрывком

Причина: рерайт включён, но исходный текст слишком короткий или непонятный.  
Решение: 🚫 Фильтры → Сетка → Канал → ✍️ Рерайт → Выключить (для новостей).

### Выходят посты с фото вместо видео

1. Проверьте настройку медиа: 🚫 Фильтры → Сетка → 📋 Каналы → [Канал] → 📹 Тип медиа
2. Установите "Только видео"
3. Если очередь уже набрана — после изменения нужно подождать следующего ночного сбора

### Как изменить время публикаций?

⏰ Расписание → ⚙️ [Сетка] → нужные параметры → **🔄 Применить расписание**

### Как включить живой режим для новостей?

⏰ Расписание → ⚙️ [Сетка] → **🔴 Вкл живой режим** → установить интервал → **🔄 Применить сейчас**

### Как добавить новую тематическую сетку?

1. 📺 → **➕ Новая сетка** → введите название
2. ⏰ → ⚙️ [Сетка] → настройте расписание → **🔄 Применить**
3. 📺 → Сетка → **➕ Добавить каналы**
4. 🔗 → добавьте источники для каждого канала
5. 🚫 → Сетка → настройте фильтры если нужно

### Изменил расписание, но ничего не изменилось

После любых изменений расписания нужно нажать **🔄 Применить расписание** в разделе ⏰.  
Остальные настройки (фильтры, источники, рерайт) применяются **сразу** без перезапуска.

### Что делать если бот завис?

⚙️ Система → **🔄 Перезапустить**

### Как посмотреть почему пост не опубликовался?

Причины пропуска записываются в базу данных. Основные:
- `media_file_missing` — медиафайл не удалось скачать
- `no_media_filter=video` — нет видео, а канал требует видео
- `no_content` — пустой пост (нет текста и медиа)
- `max_api_error` — ошибка при отправке в MAX
