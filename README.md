## 📊 Статистика

### Сегодня
Для каждой сетки и каждого канала:
- **сег.** — постов опубликовано сегодня
- **всего** — всего публикаций за всё время
- **очередь** — постов ожидают публикации

### 📅 За неделю
Публикации за последние 7 дней по каждому каналу.

### 🔗 По источникам
Топ-20 источников по количеству использованных постов.

## 📺 Каналы & Сетки

### Что такое сетка

**Сетка** — группа каналов с единым расписанием, фильтрами и настройками.  
Каждая сетка полностью **независима** от других: своё расписание, источники, рерайт, AI-провайдер.

Примеры сеток:
- **"Контент"** — каналы про моду, дом, вязание
- **"Города России"** — 174 городских новостных канала

### Главная страница раздела

Список всех сеток. Нажмите на сетку — войдёте внутрь.

**Кнопки:**
- **➕ Новая сетка** — создать сетку
- **🔍 Найти каналы MAX** — автоматически найти все каналы, где бот является администратором, и добавить новые

### 🔍 Найти каналы MAX

Бот опрашивает MAX API и показывает каналы которые ещё не добавлены в систему.  
Нажмите **➕ [Название]** рядом с нужным каналом — он добавится автоматически.  
После добавления канал нужно вручную добавить в нужную сетку.

> ⚠️ Для корректной работы бот должен быть **администратором** в канале MAX.

### Список каналов внутри сетки

Каналы отображаются по 10 на странице с кнопками ◀️ ▶️ для листания.

Для каждого канала в строке:
- **Название** — нажмите для просмотра карточки
- **✅ N** или **➕ 0** — количество источников
- **⏸** или **▶️** — пауза / возобновление
- **🗑** — удалить канал

### Карточка канала

Нажмите на название канала:
- **🔗 Источники** — источники этого канала
- **⚙️ Фильтры** — фильтры этого канала
- **⏸/▶️** — пауза / возобновление
- **🗑 Удалить канал** — удалить полностью из системы

### Добавление каналов в сетку

Сетка → **➕ Добавить каналы** → введите ключи через запятую или каждый с новой строки.

**Ключ канала** — латинское имя, например `abakan`, `novosibirsk`.  
Если вводите кириллицей (например `Абакан`) — бот автоматически транслитерирует в `abakan`.

### Убрать канал из сетки

Сетка → **➖ Убрать канал** — канал исчезнет из сетки, но **останется в системе** (источники, настройки сохранятся).

### Удаление канала

Карточка → **🗑 Удалить канал** → подтвердить.  
Канал удаляется **отовсюду**: из сеток, источников, настроек.

> ⚠️ Необратимо. Бот попросит подтверждение.

### Пауза канала

Канал на паузе пропускается при публикации. Источники и настройки сохраняются.  
Возобновить — **▶️** в строке канала или в карточке.

### Создание сетки

📺 → **➕ Новая сетка** → введите название.  
Сетка создаётся с дефолтными настройками расписания. Сразу появляется в ⏰ Расписание и 🚫 Фильтры.

### Удаление сетки

Сетка → **🗑 Удалить сетку** → подтвердить.  
Сетка удаляется из всех мест: grids, grid_settings, filters. **Каналы остаются** в системе.

## 🔗 Источники

Источники — Telegram-каналы или RSS-ленты из которых бот берёт контент.

### Навигация

Раздел сгруппирован по сеткам:

```
🔗 Источники
├── ✅ Контент (3/3)
└── ➕ Города России (6/174)
```

Сетка → канал → список источников.

### Добавление источников

1. 🔗 → Сетка → Канал → **➕ Добавить**
2. Введите username(ы) Telegram-каналов или RSS-ссылки

**Форматы ввода:**
```
cityNews_abakan
@cityNews_abakan
cityNews_abakan, another_news
https://abakan-news.ru/rss/
```
`@` писать не обязательно — бот добавит сам.

### Удаление источника

В списке источников → **🗑 Удалить** → выберите нужный.

## ⏰ Расписание

Каждая сетка имеет **своё независимое** расписание.

### Главная страница

Для каждой сетки видно текущий режим:

**Обычный режим:**
```
✅ Контент
  8:00–22:00, каждые 2 ч | Постов/день: 8
  Fetch в 23:00 МСК | Медиа: any
```

**Живой режим:**
```
🔴 Города России
  Живой режим | Fetch каждые 30 мин | Медиа: any
```

### Настройки сетки

⏰ → **⚙️ [Сетка]**

| Параметр | Описание |
|----------|----------|
| ⏱ Интервал публ. | Каждые N часов выходит 1 пост (1–12 ч) |
| 📊 Постов/день | Сколько постов набирать за один fetch (1–48) |
| 🌅 Начало | С какого часа начинать публикации (МСК) |
| 🌆 Конец | До какого часа публиковать (МСК) |
| 🕐 Час fetch | Когда собирать контент (МСК) |
| 🎬 Тип медиа | Тип медиа для всей сетки |
| ⏸/▶️ | Пауза всей сетки |

**Типы медиа:**
- **Любые** — фото, видео, текст
- **Только фото**
- **Только видео**
- **Только текст**
- **Медиа обязательно** — пропускать посты без фото/видео

### 🔴 Живой режим

Для новостных каналов — посты публикуются сразу как появляются в источнике.

**Принцип работы:**
- Каждые N минут (от 5 до 180) проверяет источники
- Берёт только посты **не старше 1 часа** от текущего момента
- Как только встречает пост старше 1 часа — прекращает листать источник
- Публикует **не более 1 поста на канал** за один цикл
- Если новостей нет — ждёт следующей проверки

**Включить:**
1. ⏰ → ⚙️ [Сетка] → **🔴 Вкл живой режим**
2. Введите интервал (рекомендуется 30 мин)
3. Нажмите **🔄 Применить сейчас**

**Выключить:** то же самое → **🔴 Выкл** → **🔄 Применить**.

### 🔄 Применить расписание

Перезапускает движок с новыми настройками.

> ⚠️ **Обязательно** нажимать после изменения расписания или режима работы.  
> Посты в очереди не теряются.

## 🚫 Фильтры

### 🌐 Глобальные фильтры

Применяются ко всем каналам, если не переопределены ниже.

**Переключатели:**

| | Описание |
|---|---|
| 📝 Требовать текст | Пропускать посты без текста |
| 🇬🇧 Блок. английский | Блокировать посты на английском |
| 🤬 Блок. мат | Блокировать нецензурную лексику |
| 📢 Блок. CAPS LOCK | Блокировать посты написанные капслоком |
| 😶 Блок. только эмодзи | Блокировать посты из одних эмодзи |
| 🔗 Блок. ссылки | Блокировать посты со ссылками |

**Числовые настройки:**

| | Описание |
|---|---|
| 📏 Мин. символов | Посты короче N пропускаются |
| 📐 Макс. символов | Посты длиннее N пропускаются (0 = без ограничений) |

| | Описание |
|---|---|

Списки поддерживают **пагинацию** — листайте кнопками ◀️ ▶️ если слов много.

Посты с этими словами публикуются **первыми** в очереди.  
Пример: `срочно`, `экстренно`, `breaking`.

Поддерживаются русский и английский языки.

```
✍️ Рерайт & AI

🤖 Провайдер: ⚡Groq (Llama)
```

- **✅/❌ [Название]** — включить / выключить рерайт для этой сетки
- **🤖 [Провайдер]** — выбрать AI-провайдер для этой сетки
- **📝 Промпт** — задать промпт для этой сетки

### Независимость сеток

Каждая сетка может использовать **свой AI-провайдер и свой промпт**.  
Например: "Контент" → Groq с коротким промптом, "Города России" → Claude с новостным промптом.

### 🤖 Выбрать провайдер / API ключи

Переходит к настройкам провайдеров.

**Доступные провайдеры:**

| Провайдер | Стоимость | Особенности |
|-----------|-----------|-------------|
| **Groq** | ✅ Бесплатно (лимиты) | Быстрый, хорошее качество |
| **Claude** (Anthropic) | 💰 Платно | Отличное качество, понимает контекст |
| **OpenAI** | 💰 Платно | Отличное качество |
| **Gemini** (Google) | ✅/💰 | Хорошее качество |

Активный провайдер отмечен **🔵**.

**Подключить платный провайдер:**
1. Нажмите на провайдер
2. **🔑 Изменить ключ** → вставьте API ключ
3. Выберите модель (опционально)
4. **🔵 Сделать активной**

### Выбор провайдера для сетки

✍️ Рерайт & AI → **🤖 [текущий провайдер]** рядом с нужной сеткой → выберите провайдера из списка.

### Промпт для сетки

✍️ Рерайт & AI → **📝 Промпт** рядом с нужной сеткой → введите промпт текстом.

**Можно использовать переменные:**
- `{text}` — оригинальный текст поста
- `{topic}` — тема канала

**Сбросить к дефолтному:** отправьте `-` (дефолт — короткая подпись для контентных каналов).

### Лимит длины рерайта

По умолчанию:
- Сетки **без кастомного промпта** — 60 токенов (короткая подпись)
- Сетки **с кастомным промптом** — 800 токенов (полный рерайт)

Можно задать явно через поле `max_tokens` в настройках сетки (конфиг).

### Рекомендации по промптам

**Для контентных каналов (мода, дом, вязание):**
> Напиши одну вдохновляющую подпись к посту. Одно предложение, максимум 10 слов. Начни со смайлика. Без хештегов и ссылок.

**Для новостных каналов:**
> Слегка отредактируй текст новости. Убери упоминания канала, ссылки, призывы подписаться. Сохрани оригинальный стиль. Добавь 1 эмодзи в начало. Выдай только готовый текст.

## Как всё работает изнутри

### Обычный режим (пример: сетка "Контент")

```
23:00 МСК — Ночной сбор (Fetch)
  → заходит в Telegram-источники
  → берёт N свежих постов на каждый канал
  → применяет фильтры (стоп-слова, медиа, OCR на картинках)
  → переписывает через AI (если рерайт включён)
  → сохраняет в очередь

08:00, 10:00, 12:00 ... 22:00 МСК — Публикация
  → берёт 1 пост из очереди для каждого канала
  → загружает медиа в MAX
  → публикует в канал
  → удаляет локальный медиафайл
```

### Живой режим (пример: сетка "Города России")

```
Каждые 30 минут:
  → проверяет источники
  → берёт только посты не старше 1 часа
  → встретил старый пост — стоп, дальше не листает
  → публикует максимум 1 пост на канал за цикл
  → если нечего публиковать — ждёт следующего цикла
```

### Старт после перезапуска

При запуске content-bot:
1. Завершает предыдущий экземпляр (если был запущен)
2. Проверяет очередь для каждой сетки
3. Если очередь пуста (для сеток без живого режима) — сразу запускает fetch
4. Запускает планировщик

### Защита от дублей

Каждый пост имеет уникальный URL. Бот помнит все увиденные URL за **30 дней** — один и тот же пост не опубликуется дважды.

### Очистка медиафайлов

Каждую ночь в **04:00 МСК** бот удаляет все медиафайлы (фото/видео) которые не привязаны к постам в очереди. Диск не засоряется.

## Частые вопросы

### Почему не публикуются посты?

Проверьте по порядку:
1. ⚙️ Система — запущен ли content-bot
2. ⏰ Расписание → сетка — активна ли (нет ⏸)
3. 📺 → Сетка → канал — не на паузе
4. 🔗 → Сетка → канал — есть ли источники
5. 📊 Статистика → канал — есть ли посты в очереди (очередь > 0)

### Посты выходят с очень коротким текстом

Рерайт обрезает текст — так задуман для контентных каналов (подпись 10 слов).  
Для новостных каналов: ✍️ Рерайт & AI → **❌ [Сетка]** — выключить рерайт для этой сетки.

### Вышел пост с рекламой или неподходящим содержимым

Добавьте стоп-слова: 🚫 Фильтры → Сетка → **🚫 Стоп-слова** → **➕ Добавить**.

### Изменил расписание — ничего не изменилось

Нажмите **🔄 Применить расписание** в разделе ⏰.  
Фильтры, источники и настройки рерайта применяются **сразу** без перезапуска.

### Как добавить новую сетку с нуля?

1. 📺 → **➕ Новая сетка** → название
2. ⏰ → ⚙️ [Сетка] → настройте расписание → **🔄 Применить**
3. Создайте каналы в MAX, добавьте бота как администратора
4. 📺 → **🔍 Найти каналы MAX** → добавьте новые каналы
5. 📺 → Сетка → **➕ Добавить каналы** → введите ключи
6. 🔗 → Сетка → добавьте источники для каждого канала
7. ✍️ Рерайт & AI → настройте провайдер и промпт для сетки
8. 🚫 Фильтры → Сетка → настройте стоп-слова

### Как включить живой режим для новостей?

⏰ → ⚙️ [Сетка] → **🔴 Вкл живой режим** → введите интервал (30 мин) → **🔄 Применить сейчас**

### Бот завис, ничего не публикует

⚙️ Система → **🔄 Перезапустить**
