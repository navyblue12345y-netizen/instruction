## Первый запуск

1. Найдите бота в Telegram и отправьте `/start`
2. Откроется **Главное меню** с кнопками разделов
3. Всё управление — только через кнопки, никаких команд вводить не нужно

> ⚠️ Бот принимает команды только от авторизованных пользователей.  
> Если вы видите "⛔" — обратитесь к суперадмину для получения доступа.

## 📊 Статистика

Показывает сколько постов опубликовано по каналам и сеткам.

### Статистика за сегодня
Главная страница раздела. Показывает по каждой сетке:
- Название канала
- Постов сегодня
- Всего постов
- Постов в очереди (ещё не опубликованы)

### 📅 За неделю
Сводка по всем каналам за последние 7 дней.

### 🔗 По источникам
Топ-20 источников по количеству опубликованных постов.

## 📺 Каналы и Сетки

Основной раздел управления структурой публикаций.

### Что такое сетка
**Сетка** — группа каналов с общими настройками расписания.  
Например: сетка "Контент" (каналы про моду, вязание, дом) и сетка "Города России" (174 городских канала).

Каждая сетка работает **полностью независимо** от других:
- своё расписание
- свои источники
- свои фильтры
- свой режим работы

### Список сеток
На главной странице раздела отображаются все сетки с количеством каналов.  
Нажмите на сетку чтобы войти в неё.

### Внутри сетки

**Список каналов** — все каналы сетки с количеством источников.  
Значки статуса:
- ✅ — активен (есть источники)
- ➕ — нет источников (нужно добавить)
- ⏸ — на паузе

**Кнопки управления сеткой:**
- 📋 Все каналы — список с настройками
- ➕ Добавить канал — добавить новый канал в сетку
- 🗑 Удалить сетку — удалить сетку (каналы не удаляются)

### Настройки канала

Нажмите на любой канал → **⚙️ Настройки**.

| Настройка | Описание |
|-----------|----------|
| 📹 Тип медиа | Что публиковать: любое / только фото / только видео / только текст / медиа обязательно |
| 📏 Мин./Макс. длина | Минимальная и максимальная длина текста поста в символах |
| 🚫 Стоп-слова | Слова из-за которых пост пропускается |
| ✍️ Рерайт | Включить/выключить рерайт для этого канала, задать свой промпт |
| 🗑 Сбросить | Удалить все индивидуальные настройки (канал вернётся к настройкам сетки) |

> ℹ️ Настройки канала имеют приоритет над настройками сетки.  
> Если у канала нет своих настроек — используются настройки сетки.

### Вкл/выкл канала

В списке каналов нажмите **⏸/▶️** рядом с каналом.  
Канал на паузе — бот пропускает его при публикации, но не удаляет.

### Создание новой сетки

1. Главная → 📺 Каналы & Сетки
2. Нажмите **➕ Создать сетку**
3. Введите название
4. Сетка создаётся с дефолтными настройками расписания — измените их в разделе ⏰ Расписание

## 🔗 Источники

Источники — Telegram-каналы из которых бот берёт контент.

### Добавление источников

1. Перейдите в нужный канал (📺 → Сетка → Канал)
2. Нажмите **➕ Добавить источники**
3. Введите username(ы) Telegram-каналов

Форматы ввода:
```
city_news_abakan
@city_news_abakan
city_news_abakan, another_channel
city_news_abakan
another_channel
(каждый с новой строки)
```

> Символ @ можно не ставить — бот добавит сам.

### Просмотр источников

Нажмите на канал → увидите список источников с количеством постов от каждого.

### Удаление источника

В списке источников нажмите **🗑** рядом с нужным источником → подтвердите.

### Поддерживаемые типы источников

| Тип | Формат | Пример |
|-----|--------|--------|
| Telegram-канал | username | `city_news` |
| RSS/Web | URL | `https://site.ru/feed/` |

## ⏰ Расписание

Управление временем публикаций. **Каждая сетка имеет своё независимое расписание.**

### Страница расписания

Показывает все сетки с их текущими настройками:
- Режим работы (обычный или 🔴 живой)
- Время публикаций
- Количество постов в день
- Время сбора контента (fetch)

### Настройки сетки

Нажмите **⚙️ [Название сетки]** для редактирования.

| Параметр | Описание |
|----------|----------|
| ⏱ Интервал | Каждые N часов публикуется один пост (1-24) |
| 📊 Постов/день | Сколько постов собирать за один fetch |
| 🌅 Начало | С какого часа начинать публикации (МСК) |
| 🌆 Конец | До какого часа публиковать (МСК) |
| 🕐 Час fetch | В какое время собирать контент (обычно ночью) |
| 🎬 Тип медиа | Тип медиа для всей сетки (можно переопределить на уровне канала) |
| ⏸/▶️ | Поставить всю сетку на паузу или включить |

### 🔴 Живой режим

Для новостных каналов где посты должны выходить сразу после появления в источнике.

**Как включить:**
1. ⏰ Расписание → ⚙️ [Сетка]
2. Нажмите **🔴 Вкл живой режим**
3. Нажмите **🔄 Применить сейчас** (обязательно!)

**Как работает:**
- Бот проверяет источники каждые N минут (настраивается)
- Если появились новые посты — публикует их сразу (до 3 за один цикл)
- Ничего нового — молчит

**⏱ Интервал fetch (мин)** — как часто проверять (5-180 минут, рекомендуется 30).

> ⚠️ После изменения любых настроек расписания нажмите **🔄 Применить расписание** — иначе изменения вступят в силу только после следующего перезапуска сервера.

### 🔄 Применить расписание

Кнопка в разделе ⏰ Расписание. Перезапускает движок публикаций с новыми настройками.  
Посты в очереди **не теряются**.

## 🚫 Фильтры

Фильтры определяют какие посты публиковать, а какие пропускать.

### Уровни фильтров

Фильтры работают по принципу приоритета (от высшего к низшему):
```
Канал → Сетка → Глобальные
```
Если у канала есть свои стоп-слова — они используются вместо глобальных.

### Глобальные фильтры

Применяются ко всем каналам если не переопределены.

**Доступные фильтры:**
- 🚫 Стоп-слова — список слов, посты с которыми пропускаются
- ✅ Белый список — если задан, публикуются ТОЛЬКО посты с этими словами
- ⭐ Приоритетные слова — посты с этими словами публикуются первыми
- 📏 Мин. длина — посты короче N символов пропускаются
- 📐 Макс. длина — посты длиннее N символов пропускаются
- 📷 Требовать фото — пропускать посты без изображений
- 🔞 Фильтр мата — автоматически блокировать посты с матом

### Фильтры сетки

В разделе 🚫 Фильтры → выберите сетку.  
Работают так же как глобальные, но только для каналов этой сетки.

### Фильтры канала

В настройках канала (📺 → Канал → ⚙️ Настройки → 🚫 Стоп-слова).

## ✍️ Рерайт

Бот автоматически переписывает текст поста перед публикацией — делает короткую подпись с эмодзи.

### Настройки рерайта

**Вкл/выкл** — включить или выключить рерайт глобально.

**Модель** — какую нейросеть использовать (настраивается в разделе 🤖 Нейросети).

**Промпт** — инструкция для нейросети. По умолчанию: написать одно вдохновляющее предложение с эмодзи, до 10 слов.

**🔄 Сбросить промпт** — вернуть промпт к заводскому.

### Рерайт для конкретного канала

📺 → Канал → ⚙️ Настройки → ✍️ Рерайт:
- Включить/выключить только для этого канала
- Задать свой промпт для этого канала

> 💡 Для новостных каналов рекомендуется **выключить рерайт** — новости должны публиковаться с оригинальным текстом.

## 🔔 Уведомления

Бот отправляет уведомление если канал долго молчит.

### Настройки

**Вкл/выкл** — включить или выключить уведомления.

**⏱ Порог молчания** — через сколько часов без публикаций прислать уведомление (1-24 ч).  
Рекомендуется: 4-6 часов.

**🌙 Ночной режим** — сколько часов ночью не беспокоить.  
Например: 6 часов → с 23:00 до 05:00 уведомления не приходят.

### Кто получает уведомления

Все пользователи с ролью **admin** и **superadmin**.

## Как всё работает изнутри

### Цикл публикации (обычный режим)

```
23:00 МСК — Fetch
  └── Бот заходит в Telegram-каналы из источников
  └── Берёт N свежих постов на каждый канал
  └── Фильтрует рекламу и нежелательное
  └── Переписывает текст через AI (если включён рерайт)
  └── Сохраняет в очередь

08:00, 10:00, 12:00... 22:00 МСК — Публикация
  └── Берёт 1 пост из очереди для каждого канала
  └── Загружает медиа в MAX
  └── Публикует пост
```

### Цикл публикации (живой режим)

```
Каждые N минут:
  └── Смотрит новые посты в источниках
  └── Если есть новые — фильтрует, сохраняет, сразу публикует
  └── Если нет новых — ничего не делает
```

### Приоритет настроек

```
Настройки канала (высший приоритет)
    ↓ если не задано
Настройки сетки
    ↓ если не задано
Глобальные настройки (низший приоритет)
```

### Защита от дублей

Каждый пост имеет уникальный URL. Бот помнит все URL которые видел за последние **30 дней** — если пост уже публиковался, он пропускается.

## Частые вопросы

### Почему бот не публикует?

1. Проверьте что канал активен (нет ⏸) — 📺 → Канал
2. Проверьте что есть источники — 🔗 Источники
3. Проверьте расписание — ⏰ Расписание
4. Проверьте статус бота — ⚙️ Система
5. Проверьте очередь в 📊 Статистика (колонка "очередь")

### Посты публикуются с пустым текстом

Возможные причины:
- Рерайт выключен, а оригинальный пост был без текста
- Источник публикует только медиа без подписи

Решение: включите рерайт для канала или добавьте другие источники.

### Как изменить время публикаций?

⏰ Расписание → ⚙️ [Сетка] → нужные параметры → **🔄 Применить расписание**

### Как полностью остановить публикации?

- Одна сетка: ⏰ Расписание → ⚙️ [Сетка] → ⏸ Поставить на паузу
- Один канал: 📺 → Канал → ⏸

### Как добавить новую тематическую сетку?

1. 📺 → ➕ Создать сетку → введите название
2. ⏰ → ⚙️ [Сетка] → настройте расписание
3. 📺 → Сетка → ➕ добавьте каналы
4. 🔗 → добавьте источники для каждого канала
5. ⚙️ → 🔄 Перезапустить

### Изменение настроек не работает

После изменения расписания нужно нажать **🔄 Применить расписание** в разделе ⏰.  
Остальные настройки (фильтры, источники, рерайт) применяются сразу без перезапуска.

### Что делать если бот завис?

⚙️ Система → 🔄 Перезапустить
